# funpay_parser

`Telegram Bot` для реализации функции отслеживание данных на сайте FunPay

Вводятся данные для фильтрации, после чего отслеживаются.
При измененеие выдает уведомлением в личное сообщение Telegram


Команды бота:

`/start`
`/stop`

Бибилотеки:

- `requests`

- `aiogram`

- `BeautifulSoup`

- `lxml`
